## 不吸收内容

列出不适合 Trellis 的内容：

- 与 Trellis 任务/需求追踪冲突的状态模型。
- 依赖特定私有工具或平台的实现。
- 过重运行时。
- license 或安全边界不清晰的文本/脚本。
